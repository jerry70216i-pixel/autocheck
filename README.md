# 工程照片查驗表自動化系統

目前只實作 V0：本機選取照片、預覽照片、輸入查驗說明、匯出單一固定 HTML 表格。

## 啟動方式

不需要安裝套件。請在專案根目錄執行：

```powershell
python -m http.server 8000 -d public
```

然後開啟：

```text
http://127.0.0.1:8000
```

若系統沒有 `python`，也可以直接用瀏覽器開啟 `public/index.html`。

## V0 使用方式

1. 填寫工程名稱、日期、位置、查驗項目。
2. 選取多張工程照片。
3. 在照片預覽區檢查順序。
4. 視需要調整照片順序。
5. 為每張照片輸入查驗說明。
6. 點選「重新產生表格預覽」確認固定表格。
7. 點選「匯出 HTML 表格」下載含照片的表格。

## V0 限制

- 不做 AI。
- 不做 RAG。
- 不做帳號登入。
- 不做資料庫。
- 不做後端 API。
- 不做多表格模板。
- 不做多格式輸出。

## 手動測試步驟

1. 啟動本地服務並開啟頁面。
2. 選取 2 張以上照片。
3. 確認照片縮圖出現在預覽區。
4. 輸入每張照片的查驗說明。
5. 調整其中一張照片順序。
6. 點選「重新產生表格預覽」。
7. 確認表格內有照片、檔名與說明。
8. 點選「匯出 HTML 表格」。
9. 開啟下載的 HTML，確認照片與說明仍可閱讀。
