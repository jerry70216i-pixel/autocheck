## V2 Word 範本輸出

狀態：已完成，並已通過人工驗收。

V2 使用 `.docx` Word 範本輸出正式查驗表。

### 正式方案

- 使用 `.docx` Word 範本輸出正式查驗表。
- 不再使用 HTML 表格模擬 Word。
- 不使用 `.doc` 舊格式作為正式模板。
- 不新增 `package.json`。
- 不新增 Node build flow。
- 純前端靜態專案。
- 使用 `public/vendor/` 內建 PizZip、Docxtemplater、ImageModule。

### 正式模板

- `templates/self-check-template.docx`
- `templates/supervision-inspection-template.docx`

### 自主檢查

- 最多 30 張
- 每頁 3 張
- 最多 10 頁
- 圖片尺寸 `[460, 280]`
- 未使用頁面需移除

### 監造查驗

- 最多 20 張
- 每頁 2 張
- 最多 10 頁
- 圖片尺寸 `[480, 300]`
- 未使用頁面需移除

### V2 已驗收項目

- 正式模板可載入。
- Word `.docx` 可產生與下載。
- placeholder 不殘留。
- 未使用頁面可移除。
- 自主檢查 30 張輸出版面可接受。
- 監造查驗 20 張輸出版面可接受。
- 圖片尺寸符合目前人工驗收設定。

### V2 不做事項

- 不做 AI。
- 不做 RAG。
- 不做雲端資料庫。
- 不做伺服器端輸出。
- 不做 N / E / Z 任意欄位擴充。
- 不新增 `package.json`。
- 不新增 Node build flow。
- 不使用 `.doc` 舊格式作為正式模板。

## V3：AI 輔助與後續擴充

V3 僅能在 V2 Word 範本輸出穩定後評估，不併入目前 V2 範圍。
