# V1 Checklist

V1 完成前必須確認以下項目：

## 工項資料

- [ ] 可以建立預設工項資料
- [ ] 工項資料由 repository / storage 層提供
- [ ] UI 不直接操作 IndexedDB 或 localStorage

## 查驗說明範本

- [ ] 可以建立預設查驗說明範本
- [ ] 每個工項可以對應預設查驗說明
- [ ] 查驗說明範本由 repository / storage 層提供

## 照片操作

- [ ] 每張照片可以選擇工項
- [ ] 選擇工項後可以自動帶入預設說明
- [ ] 使用者可以修改說明
- [ ] 未填工項或說明時仍可匯出

## 匯出流程

- [ ] 匯出時可以輸出照片與說明
- [ ] 即使沒有選工項，也不破壞 V0 照片匯出流程
- [ ] 即使沒有查驗說明，也不破壞 V0 照片匯出流程

## 本機資料儲存

- [ ] 重新整理頁面後，工項與說明資料仍保留
- [ ] 資料儲存邏輯集中在 repository / storage 層
- [ ] UI 不直接操作 IndexedDB 或 localStorage
- [ ] 本機資料儲存失敗時，基本照片匯出流程不應崩潰
