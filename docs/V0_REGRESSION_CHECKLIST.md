# V0 Regression Checklist

V1 或後續版本修改後，必須確認以下 V0 功能仍正常：

## 照片流程

- [ ] 可以選取 / 上傳照片
- [ ] 可以顯示照片預覽
- [ ] 照片順序不會被錯亂
- [ ] 不支援的檔案格式會被排除或提示

## 匯出流程

- [ ] 可以匯出表格
- [ ] 照片可以正確放入表格
- [ ] 匯出檔案可以正常開啟
- [ ] 沒有照片時不會產生錯誤崩潰
- [ ] 沒有選工項也可以匯出
- [ ] 沒有查驗說明也可以匯出
- [ ] 本機資料儲存失敗時，基本照片匯出流程不應崩潰

## V1 修改後仍需符合

- [ ] 工項與說明不得成為 V0 匯出的必要條件
- [ ] 新增說明欄位後，照片位置仍正確
- [ ] 資料儲存失敗時，仍可手動操作並匯出
