# V2 Checklist

V2 完成前必須確認以下項目：

## V0 / V1 保護

- [ ] 可以選取 / 上傳多張照片
- [ ] 可以顯示照片預覽
- [ ] 照片順序正常
- [ ] 可以移除照片
- [ ] 每張照片可以選工項
- [ ] 選工項後會帶入預設說明
- [ ] 可以手動修改說明
- [ ] 不選工項也可以匯出
- [ ] 不填說明也可以匯出

## V2 匯出格式

- [ ] 可以選擇「自主檢查」格式並匯出
- [ ] 可以選擇「監造查驗」格式並匯出
- [ ] 兩種格式的匯出檔案都可以用 Word 開啟
- [ ] 自主檢查格式接近範本：一頁 3 組，左圖右日期 / 位置 / 說明
- [ ] 監造查驗格式接近範本：一頁 2 組，上方公司 / 工程 / 拍攝項目，左圖右日期 / 位置 / 說明
- [ ] 兩種格式內照片都正確
- [ ] 兩種格式內說明都正確
- [ ] 兩種格式檔名正確
- [ ] 自主檢查位置欄使用全域施工位置，不使用照片檔名
- [ ] 監造查驗位置欄使用全域施工位置，不使用照片檔名
- [ ] 施工位置空白時，位置欄留空
- [ ] UI 顯示「V2 Word 範本版」

## 架構

- [ ] 匯出格式邏輯在 exporter / template adapter 層
- [ ] `app.js` 不直接塞滿不同表格格式判斷
- [ ] 未新增 dependency
- [ ] 未直接修改原始 Word 範本
- [ ] repository / IndexedDB 失敗時，V0 / V1 基本流程仍可運作
